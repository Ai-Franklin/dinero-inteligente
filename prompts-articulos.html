Prompts para crear artículos completos de educación financiera

Usa estas plantillas para producir contenido en español para Dinero Inteligente: artículos largos, tablas, diagramas, gráficos e imágenes educativas.

Cómo usar esta página

Elige un tema, copia el prompt maestro, reemplaza los campos entre corchetes y pégalo en tu IA favorita. Después usa los prompts visuales para crear la imagen principal, una infografía y gráficos para redes.

Prompt maestro para artículos financieros

Este prompt está pensado para artículos de 1,500 palabras o más, con explicación sencilla, enfoque de principios y materiales visuales. No intenta copiar a Ray Dalio; usa ideas parecidas: sistemas, ciclos, causa y efecto, riesgo y decisiones racionales.

Actúa como escritor experto en educación financiera para principiantes de habla hispana.

Quiero que escribas un artículo completo para el blog "Dinero Inteligente".

TEMA: [escribe aquí el tema]
PALABRA CLAVE PRINCIPAL: [ejemplo: cómo salir de deudas]
PALABRAS CLAVE SECUNDARIAS: [3 a 6 palabras clave relacionadas]
AUDIENCIA: personas principiantes que quieren mejorar su dinero con explicaciones simples.

Estilo editorial:
- Español claro, humano y práctico.
- Explica como si la persona no supiera nada de finanzas.
- Usa un enfoque inspirado en principios financieros: causa y efecto, ciclos, deuda, ahorro, inversión, riesgo, diversificación y toma de decisiones racional.
- No copies la voz exacta de ningún autor famoso.
- Evita promesas de riqueza rápida.
- Evita sonar como IA: no uses frases infladas como "en el dinámico mundo financiero", "es crucial destacar" o "sin más preámbulos".
- Usa ejemplos cotidianos: salario, renta, supermercado, tarjeta de crédito, préstamo, ahorro de emergencia e inversión básica.

Entrega el contenido en este orden:

1. Título SEO
- Máximo 65 caracteres.
- Incluye la palabra clave principal de forma natural.

2. Meta descripción
- Máximo 155 caracteres.
- Clara, atractiva y sin exageraciones.

3. Introducción
- Empieza con un problema real de dinero.
- Explica por qué importa.
- Promete una solución educativa y práctica.

4. Artículo completo en Markdown
- Mínimo 1,500 palabras.
- Usa H2 y H3.
- Explica conceptos difíciles con lenguaje simple.
- Incluye ejemplos numéricos.
- Incluye al menos una tabla.
- Incluye enlaces internos sugeridos a otros artículos de Dinero Inteligente.

5. Sección obligatoria: "El principio detrás de este problema"
Explica la lógica causa-efecto. Ejemplo:
Si gastas más de lo que ganas, dependes de deuda.
Si dependes de deuda, pagas intereses.
Si pagas demasiados intereses, pierdes libertad.
Si reduces deuda y aumentas activos, recuperas control.

6. Ejemplo con números
Crea una tabla realista con cifras hipotéticas. Aclara que son ejemplos educativos.

7. Diagrama Mermaid
Crea un diagrama simple en este formato:
```mermaid
flowchart TD
A[Ingresos] --> B[Gastos esenciales]
A --> C[Ahorro]
A --> D[Pago de deudas]
C --> E[Fondo de emergencia]
D --> F[Menos intereses]
F --> G[Más libertad financiera]
```

8. Tabla para gráfico
Incluye una tabla que pueda convertirse en gráfico. Por ejemplo:
- pago mínimo vs pago acelerado,
- ahorrar 5%, 10% y 20%,
- invertir temprano vs invertir tarde,
- gastos hormiga antes y después.

9. Prompts de imágenes
Crea 4 prompts:
- imagen principal del artículo,
- imagen conceptual interna,
- diagrama visual,
- infografía vertical para Pinterest.
Cada prompt debe describir estilo, colores, composición y elementos visuales.
Usa estilo profesional educativo, colores azul oscuro, verde, blanco y dorado suave.
Evita texto pequeño difícil de leer.

10. Qué puedes hacer esta semana
Da 5 a 7 acciones simples que alguien con poco dinero pueda hacer.

11. Errores que debes evitar
Explica 5 a 7 errores frecuentes.

12. La idea en pocas palabras
Resume en 5 bullets claros.

13. Preguntas frecuentes
Incluye 5 preguntas frecuentes pensadas para SEO.

14. Nota educativa
Añade una nota breve: este contenido es educativo y no sustituye asesoría financiera profesional.

Reglas finales:
- No inventes datos económicos actuales.
- Si usas porcentajes, aclara que son ejemplos hipotéticos.
- No recomiendes productos específicos sin explicar riesgos.
- Escribe con ritmo humano: frases cortas mezcladas con explicaciones más largas.
- Prioriza claridad antes que palabras elegantes.

Prompt para imagen principal

Úsalo en generadores como DALL-E, Flux, Ideogram, Leonardo AI o Midjourney.

Crea una imagen principal para un artículo de educación financiera en español.

Tema del artículo: [tema]

Estilo visual:
- profesional, limpio y educativo,
- colores azul marino, verde, blanco y dorado suave,
- iluminación clara,
- estilo infografía moderna con elementos simples,
- personas latinas o hispanas si aparecen personajes,
- ambiente de confianza, calma y aprendizaje.

Composición:
- una persona revisando su dinero con una libreta o laptop,
- iconos simples de ahorro, deuda, inversión y presupuesto,
- una flecha visual que muestre progreso financiero,
- fondo ordenado y sin distracciones.

Evita:
- billetes volando de forma exagerada,
- promesas visuales de riqueza rápida,
- texto pequeño ilegible,
- logos de bancos reales,
- apariencia de publicidad engañosa.

Formato: imagen horizontal 16:9 para portada de blog.

Prompt para infografía vertical

Diseña una infografía vertical en español para principiantes sobre:

TEMA: [tema]

Objetivo:
Explicar el tema de forma clara, visual y práctica.

Estructura de la infografía:
1. Título principal claro.
2. El problema financiero.
3. El principio causa-efecto.
4. Ejemplo numérico simple.
5. Pasos prácticos.
6. Resultado esperado realista.

Estilo:
- fondo claro,
- colores azul marino, verde, blanco y dorado suave,
- iconos financieros simples,
- máximo 6 bloques de información,
- tipografía grande y legible,
- diseño limpio para Pinterest y redes sociales.

No incluir:
- promesas de dinero rápido,
- mucho texto pequeño,
- logos de bancos reales,
- cifras que parezcan garantía.

Formato: vertical 9:16.

Prompt para diagrama o gráfico

Crea un diagrama educativo simple en español sobre:

TEMA: [tema]

Debe mostrar la relación causa-efecto entre decisiones financieras y resultados.

Formato sugerido:
- flujo con flechas,
- 5 a 7 pasos máximo,
- etiquetas cortas,
- colores azul marino, verde y blanco,
- iconos simples para ingreso, gasto, deuda, ahorro, inversión y libertad financiera.

Ejemplo de lógica:
Ingresos -> Gastos -> Ahorro -> Fondo de emergencia -> Menos deuda -> Más control financiero

El diseño debe ser limpio, fácil de leer en móvil y útil dentro de un artículo de blog.

Plantilla rápida para tabla de datos

Las tablas ayudan a que el artículo se sienta práctico. También sirven para crear gráficos después.

Nota: cifras hipotéticas para explicar el concepto. No son una recomendación personalizada.

Checklist antes de publicar

SEOTítulo menor de 65 caracteres, meta descripción, palabra clave natural y FAQ.
UtilidadEjemplo numérico, pasos concretos y errores comunes.
VisualesImagen principal, infografía, tabla y diagrama Mermaid.
ConfianzaSin promesas de riqueza rápida. Incluye nota educativa.
MonetizaciónMarca espacios para afiliados solo si el producto ayuda al lector.
EnlacesAgrega enlaces internos a otros artículos del blog.
